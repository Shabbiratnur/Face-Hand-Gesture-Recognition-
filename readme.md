# Prerequisites for this project:

1. Python – 3.x (we used Python 3.8.8 in this project)

2. OpenCV – 4.5
    Run “pip install opencv-python” to install OpenCV.

3. mediapipe-0.8.5
    Run "pip install mediapipe" to install mediapipe.


# Run the program

`python test.py`
    
# To exit the app press q.
